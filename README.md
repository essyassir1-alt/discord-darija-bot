# 🗣️ Discord Darija Voice Bot

A Discord bot that joins voice channels, listens to users, and responds in Moroccan Darija (الدارجة المغربية) using OpenAI's Whisper, GPT, and TTS.

## ✨ Features

- 🎙️ Join voice channels and listen to users
- 🔊 Automatic speech recognition (STT) with Whisper
- 🤖 AI responses in Moroccan Darija using GPT
- 📢 Text-to-speech (TTS) replies in voice channel
- 💬 Natural conversation with context memory
